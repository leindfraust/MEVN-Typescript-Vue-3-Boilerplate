# A boilerplate of latest MEVN technologies

This is a boilerplate for a full stack development leveraging the following tech stack: MongoDb, Express.js, Vue.js and Node.js.

## Inclusions

### Vue.js

- Router
- Pinia
- Bulma
- Typescript

PS: This uses Webpack not Vite.

### Node.js

- CORS
- Mongoose
- Morgan
- Connect-history-api-fallback
- Helmet

PS: Node.js will be using ES Module or "type": "module"

## Development

### Install dependencies

#### Server(root directory)

``` npm install ```

#### Client

``` cd client && npm install ```

## Command Scripts

### Root Directory

***

#### Server only

``` npm run server ```

#### Client only

``` npm run client ```

#### Server and client development

``` npm run dev ```

#### Production server

``` npm run start ```

#### Render deployment

``` npm run render ```

***

### Client Directory

#### Client development

``` npm run serve ```

##### Build for production

``` npm run build ```

##### Lint

``` npm run lint ```
